Utility for setting up a socket (or pair of sockets) from a text-based description.

Currently this is only used for UDP multicast.

Parameters:
descrhold the information needed to create the socket(s).
loggershould be used for reporting errors, printf-style.
logdatmust be passed as first argument to logger().
getboundis callback for getting already-bound sender socket, should return -1 if none.
getbounddatis passed as first argument to getbound
sockswill be filled in with the pair of socket file descriptors.
Returns:
0 for success, -1 for error.
