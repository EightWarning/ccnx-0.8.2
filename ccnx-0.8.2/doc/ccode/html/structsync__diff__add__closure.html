Detailed Description

sync_diff_add_closure is called from the differencing operation to note a name difference for sdd.

If name == NULL then there are no more names to be produced, and there should be no future references to sdd. The data is private to the closure, and the client owns all storage in the closure, except for sync_diff_data.
